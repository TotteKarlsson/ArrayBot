MOT_ButtonModes MOT_ButtonParameters::buttonMode

Defines the mode of operation of the device buttons

The buttons on a TCube can be defined to either jog or move to a preset position.

1Jog the device according to the MOT_JogParameters
2Move to the preset button positions.
