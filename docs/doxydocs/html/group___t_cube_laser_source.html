Detailed Description

This section details the Structures and Functions relavent to the TCube Laser Source
For an example of how to connect to the device and perform simple operations use the following links:

  • Example of using the Thorlabs.MotionControl.TCube.LaserSource.DLL from a C or C++ project.
    This requires the DLL to be dynamical linked.
  • Example of using the Thorlabs.MotionControl.TCube.LaserSource.DLL from a C# project
    This uses Marshalling to load and access the C DLL.

The Thorlabs.MotionControl.TCube.LaserSource.DLL requires the following DLLs

  • Thorlabs.MotionControl.DeviceManager.

Enumeration Type Documentation

enum LS_DisplayUnits : unsigned short

Values that represent Laser Display Units.

Enumerator
LS_mAmps 

Dislay shows the output current in mAmps.

LS_mWatts 

Dislay shows the output power in mWatts.

LS_mDb 

Dislay shows the output power in dBm.

enum LS_InputSourceFlags : unsigned short

Values that represent Laser Input Source Flags.

Enumerator
LS_SoftwareOnly 

Input is read from software only.

LS_ExternalSignal 

Input is read from software or External signal.

LS_Potentiometer 

Input is read from software or Potentiometer.
