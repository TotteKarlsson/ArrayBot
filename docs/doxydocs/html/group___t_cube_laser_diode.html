Detailed Description

This section details the Structures and Functions relavent to the TCube Laser Diode
For an example of how to connect to the device and perform simple operations use the following links:

  • Example of using the Thorlabs.MotionControl.TCube.LaserDiode.DLL from a C or C++ project.
    This requires the DLL to be dynamical linked.
  • Example of using the Thorlabs.MotionControl.TCube.LaserDiode.DLL from a C# project
    This uses Marshalling to load and access the C DLL.

The Thorlabs.MotionControl.TCube.LaserDiode.DLL requires the following DLLs

  • Thorlabs.MotionControl.DeviceManager.

Enumeration Type Documentation

enum LD_DisplayUnits : unsigned short

Values that represent Laser Display Units.

Enumerator
LD_ILim 

Dislay shows the Max Laser Diode Current in mAmps.

LD_ILD 

Dislay shows the Laser Diode Current in mAmps.

LD_IPD 

Dislay shows the Photodiode current in mA.

LD_PLD 

Dislay shows the Ouput Power in uW.

enum LD_InputSourceFlags : unsigned short

Values that represent Laser Input Source Flags.

Enumerator
LD_SoftwareOnly 

Input is read from software only.

LD_ExternalSignal 

Input is read from software or External signal.

LD_Potentiometer 

Input is read from software or Potentiometer.

enum LD_POLARITY

Values that represent LD_POLARITY.

Enumerator
LD_CathodeGrounded 

Configure the device for Cathode Grounded.

LD_AnodeGrounded 

Configure the device for Anode Grounded.

enum LD_TIA_RANGES

Values that represent LD_TIA_RANGES.

Enumerator
LD_TIA_10uA 

The device TIA range is set to 10uA.

LD_TIA_100uA 

The device TIA range is set to 100uA.

LD_TIA_1mA 

The device TIA range is set to 1mA.

LD_TIA_10mA 

The device TIA range is set to 10mA.
