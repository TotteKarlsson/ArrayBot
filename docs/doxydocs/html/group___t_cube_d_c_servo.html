Detailed Description

This section details the Structures and Functions relavent to the TCube DC Servo
For an example of how to connect to the device and perform simple operations use the following links:

  • Example of using the Thorlabs.MotionControl.TCube.DCServo.DLL from a C or C++ project.
    This requires the DLL to be dynamical linked.
  • Example of using the Thorlabs.MotionControl.TCube.DCServo.DLL from a C# project
    This uses Marshalling to load and access the C DLL.

The Thorlabs.MotionControl.TCube.DCServo.DLL requires the following DLLs

  • Thorlabs.MotionControl.DeviceManager.
