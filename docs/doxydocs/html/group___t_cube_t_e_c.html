Detailed Description

This section details the Structures and Functions relavent to the TEC
For an example of how to connect to the device and perform simple operations use the following links:

  • Example of using the Thorlabs.MotionControl.TCube.TEC.DLL from a C or C++ project.
    This requires the DLL to be dynamical linked.
  • Example of using the Thorlabs.MotionControl.TCube.TEC.DLL from a C# project
    This uses Marshalling to load and access the C DLL.

The Thorlabs.MotionControl.TCube.TEC.DLL requires the following DLLs

  • Thorlabs.MotionControl.DeviceManager.

Enumeration Type Documentation

enum TC_DisplayModes : unsigned short

Values that represent the display / output modes.

Enumerator
TC_ActualTemperature 

Display actual temperature.

TC_TargetTemperature 

Display target temperature.

TC_TempDifference 

Display difference in temperature between actual and target.

TC_Current 

Display actual current.

enum TC_SensorTypes : unsigned short

Values that represent the available sensor types.

Enumerator
TC_Transducer 

Using AD59x IC transducer sensor.

TC_TH20kOhm 

Using 20kOhm thermistor.

TC_TH200kOhm 

Using 200kOhm thermistor.
