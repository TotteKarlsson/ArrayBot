MOT_LimitSwitchModes MOT_LimitSwitchParameters::anticlockwiseHardwareLimit

Defines the anticlockwise hardware limit.

The anticlockwise hardware limit

0x01Ignore limit switch
0x02Make on contact.
0x03Break on contact.
0x04Makes on contact when homing
0x05Break on contact when homing.
0x06Reserved for PMD brushless servo controllers.
0x80Switch mode when using a rotational stage.
