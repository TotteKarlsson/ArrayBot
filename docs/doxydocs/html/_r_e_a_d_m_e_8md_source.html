1 PyAPT
2 =====
3 Controlling Thorlabs APT using Python code using APT.dll and APT.lib, bypassing the activex controls.
4 It even comes with a simple GUI!
5 
6 **Installation**
7 
8 Obtain APT.dll and APT.lib from Thorlabs directly, or from the ```APTDLLPack/DLL/``` folder, picking the correct version (```x86``` or ```x64```)
9 Place APT.dll, APT.lib and PyAPT.py in the folder from which the scripts will be executed, or in the main add-on folder for python.
10 You also require ThorlabsAPT software (obtainable from [ThorLabs APT Software page](http://www.thorlabs.us/software_pages/ViewSoftwarePage.cfm?Code=APT) )
11 
12 **Running the code**
13 
14 Ensure correct version (32 or 64 bit) APT.dll and APT.lib are in the folder containing the python script.
15 [```Example.py```](Example.py) contains an example of how to load the module, control the motor, and close the object.
16 
17 **Using the GUI**
18 
19 ![](https://cloud.githubusercontent.com/assets/5076314/7212786/91687864-e521-11e4-89c9-4a843e87a142.png)
20 
21 GUI is implemented using QT. First enter the serial number, and press the connect button.
22 The GUI will attempt to connect to the APT motor, and probe for it's current position and velocity. If sucessfull, those values will be updated in the respective boxes.
23 Pushbuttons under the position box are for movement in microns.
