KMOT_TriggerPortMode KMOT_TriggerConfig::Trigger1Mode

The trigger 1 mode.

The trigger 1 operating mode:

0Trigger disabled
1Trigger Input - General purpose logic input
2Trigger Input - Move relative using relative move parameters
3Trigger Input - Move absolute using absolute move parameters
4Trigger Input - Perform a Home action
10Trigger Output - General purpose output (set using MOT_SET_DIGOUTPUTS)
11Trigger Output - Set when device moving
12Trigger Output - Set when at max velocity
13Trigger Output - Set when at predefine position steps
